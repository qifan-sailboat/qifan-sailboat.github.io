Biography

I’m a Security Researcher at Palo Alto Networks working on network security, with a focus on the security of the Domain Name System (DNS). I build automated analysis tools — combining fuzzing, protocol analysis, and formal methods — that have surfaced CVE-recognized vulnerabilities in widely deployed DNS resolvers. I also study security and privacy risks in deployed machine-learning systems.
I received my Ph.D. in Computer Engineering from UC Irvine in 2025, advised by Zhou Li, and my B.Eng. from ShanghaiTech University in 2020. [CV]
